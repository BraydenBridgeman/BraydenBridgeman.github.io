ABOUT ME

My name is Brayden Bridgeman. I take photos. I build websites.
I am currently in the process of completing the Eleven Fifty Academy course in Web Development.
My coding experience is minimal to start, but it starts at a young age.
Ever since MySpace allowed you to edit HTML, my intrest was
solidified in design and creating content that was appealing to me.

Thanks Tom!

I have a Bachelor's Degree from Ball State University where I studied video production.
Worked with ESPN and the Big Ten Network to broadcast and film live sports production,
photographed famous artists for a media company in Indianapolis, and can now say I created my own website from scratch!
Learning code has been a challenging but engaging experience for me.
I am developing my skills as I progress throughout this program and having a blast while doing it!

Mars Weather API project was created using VSC.
Created using NASA API to pull data about their Mars InSight Rover
